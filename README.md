# starwars-characters

# Requirements

* Node >= v8.10.0 

# Stack

* Stylus
* PWA
* Offline
* List all characters
* Pagination
* Webpack
* React
* ES6 and ES7

# Install dependencies

`$ npm install`

# Run the prejct

`$ npm start`

# View the project

Open in your browser the following url `http://localhost:3000`
